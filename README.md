# Solve9 — Think in Grids

Solve9 is a modern Sudoku mobile application built with React Native and Expo.
The app focuses on clarity, logical problem solving, and smooth user interactions,
while maintaining a clean and professional visual design.

---

## Overview

Solve9 is designed as a production-quality mobile application and a learning-driven
project. It combines a polished user interface with strong state management and
algorithmic foundations, making it suitable for real users as well as technical evaluation.

---

## Features

- 9×9 Sudoku grid with correct 3×3 sub-grid grouping
- Cell selection with row, column, and box highlighting
- Same-number highlighting across the grid
- Fixed (immutable) puzzle cells
- Number input row with remaining-count indicator
- Material-style layout and animations
- Clean typography using the Rubik font
- Mobile-first, touch-optimized design

---

## Getting Started

### Prerequisites
- Node.js (LTS)
- Expo CLI
- Expo Go (for running the app on a physical device)

### Installation

```bash
npm install
npx expo start
```

Scan the QR code using the Expo Go application.
